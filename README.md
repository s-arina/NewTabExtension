## New Tab

A new tab extension for Chrome made using React.

![new tab demo](https://github.com/s-arina/NewTabExtension/blob/main/images/newTab.gif)

### Features

- Date/Time
- Weather (toggle between Fahrenheit/Celcius)
- Notepad
- Multiple background options
